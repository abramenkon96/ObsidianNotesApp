# Руководство по контрибьютингу 🤝

Спасибо, что решили помочь в развитии **Obsidian Notes**! 

## Как вы можете помочь:
- **Баг-репорты:** Если вы нашли ошибку, создайте Issue с подробным описанием и шагами для воспроизведения.
- **Предложения:** Мы открыты для новых идей! Пишите их в раздел Issues.
- **Код:** 
    1. Сделайте Fork.
    2. Создайте ветку (`git checkout -b feature/AmazingFeature`).
    3. Закоммитьте изменения (`git commit -m 'Add some AmazingFeature'`).
    4. Пушните в ветку (`git push origin feature/AmazingFeature`).
    5. Откройте Pull Request.

## Стандарты кода:
- Используйте Kotlin 2.0.
- Придерживайтесь MVVM.
- Всегда проверяйте совместимость YAML-метаданных.
